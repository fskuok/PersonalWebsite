PROBLEM

In recent years, many manufacturers produce various large screen cell phones. Large screens bring users better visual experience, however it is inconvenient to control. Especially in situations when users have only one hand available.

Anther problem with cell phones is when user using cell phones lying on the side or reclining at a sofa, the gyroscope will mislead the smart phone to change between horizontal interface and vertical interface.

CONCEPT

Let phones feel how people hold them

then

respond with an ergonomic interface

INTERACTION STUDY: THE THREE ZONES

Mobile phone screen into 3 areas for different comfortableness for control .These areas are guideline of designing ergonomics interface.

UI prototypes was designed base on the 3 areas guideline. Users ability test were conducted, and interfaces are iterated to be improved according to feed backs.
